nginx 常用命令

2021-04-27 杂项 nginx, server 评论

nginx 常用命令

  • sudo nginx // 启动 nginx
  • sudo nginx -s reload // 重启 nginx
  • sudo nginx -s stop // 关闭 nginx
  • ps -ef | grep nginx // 查看查看nginx进程
  • sudo kill -TERM 主进程号 // 杀死进程

linux 常用命令

  • cp [-r]
  • rm
  • open
  • pwd
  • clear reset
